# 2048 pure Html, css and js made

An individual project based on the recreation of the classic 2048 game in plain HTML, CSS and JavaScript.
No libraries or structures, it simply consists of clean hand crafted front-end code.

The game includes:
- Smooth up down animations  
- Dark/Light toggle for best exp.
- inspired by milkyway (navy, blues, cream, peach)   
- localStorage  

## Features

### Classic 2048 Mechanics
- merge tiles to reach **2048** or more.
- **Arrow Keys**, **WASD**are supported.
- Tiles animate smoothly with a small “pop” on spawn.

1. download or clone my repository.

3. Play instant - no more steps needed.

- Chrome  
- Firefox  
- Safari  
- Edge
